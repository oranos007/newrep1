Introduction à Git

Git est un système de contrôle de version distribué qui permet aux développeurs de suivre les changements dans leur code et de collaborer efficacement.

Concepts Clés

  • Répertoire: Stocke les fichiers et l'historique d'un projet.
  • Commit: Enregistre les changements avec un message.
  • Branche: Une version séparée pour le développement.

Commandes de Base

  • git init: Initialiser un répertoire.
  • git add: Préparer les changements.
  • git commit: Enregistrer les changements.
  • git push: Envoyer les changements à un répertoire distant.

Flux de Travail Git

Git propose plusieurs flux de travail, notamment le branching par fonctionnalité et Gitflow, pour gérer le code efficacement.
